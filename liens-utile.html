Vous trouverez ici une liste non-exhaustive de logiciels améliorant l'interactivité avec des serveurs distants, comme ceux loués chez nanoClouD®. La plupart permettent de vous connecter de façon cryptée à nos serveurs, en utilisant le protocole SSH, ce que nous recommandons. Pour vous simplifer l'utilisation de ce protocole, nous veillons d'ailleurs à installer un serveur SSH dans toutes nos configurations, aussi bien sous Linux que sous Windows.

Putty

Ce logiciel permet de se connecter, en mode console, depuis un environnement Windows vers une machine disposant d'un serveur SSH.

WinSCP

À l'instar de Putty, WinSCP peut être utilisé pour effectuer des transferts sécurisés entre votre poste client local sous Windows et votre serveur distant. WinSCP dispose par ailleurs d'un interface graphique facilitant le transfert de fichiers entre machines.

FileZilla

Ce programme est une sorte de couteau suisse des transferts FTP, avec notamment le support de SFTP qui permet aussi un transfert de fichiers entièrement crypté et sécurisé. Par ailleurs, FileZilla, logiciel open source, est disponible aussi bien sous Windows, que sous Linux ou MacOSX.

NoMachine NX

Pour pouvoir visualiser une session graphique sur un de nos serveurs Linux, sous Gnome, KDE ou XFCE, ce programme est particulièrement intéressant. Il privilégie une connexion cryptée via SSH, pour transférer la session distante, tout en compressant les données échangées, de façon à fournir une interactivité confortable, même lorsqu'on ne dispose que d'une ligne ayant un débit moyen. Le protocole NX peut également vous permettre de connecter votre imprimante locale à votre serveur distant, ce qui en fait une expérience utilisateur intéressante pour un poste virtuel en ligne. Ce logiciel n'est pas Open Source mais il est disponible gratuitement en version client illimitée sous Windows
